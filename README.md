# 喜乐贩卖机 - 扭蛋机抽奖活动 Landing Page

一个精美的单页活动页面，展示扭蛋机抽奖活动。

## 功能特性

- 🎨 **3D透视网格背景** - 动态网格线营造空间感
- 🎯 **居中主视觉** - 扭蛋机作为视觉中心
- 🎬 **交互动画** - 点击触发扭蛋机动画视频播放
- 📱 **响应式设计** - 完美适配桌面端和移动端
- ✨ **精美UI** - 现代化的设计风格

## 文件结构

```
Capsule-machine/
├── index.html          # 主页面结构
├── styles.css          # 样式文件
├── script.js           # 交互逻辑
├── niudanji.mp4        # 扭蛋机动画视频
└── README.md           # 说明文档
```

## 页面布局

### 1. 背景层
- 白色/浅色渐变背景
- 3D透视网格线动画效果

### 2. 顶部信息区
- **左上角**：活动标题「喜乐贩卖机」+ 装饰圆点
- **右上角**：活动状态「限时扭蛋」+ 日期范围

### 3. 中央主视觉区
- 扭蛋机插画（CSS绘制）
- 点击后播放 `niudanji.mp4` 动画视频

### 4. 两侧辅助文案
- **左侧**：纵向文案「2026 YEARS」+ 活动说明
- **右侧**：竖排大字「扭蛋机」+ 装饰花括号

### 5. 底部行动区
- CTA按钮：「参加扭蛋机抽奖活动」
- 英文辅助说明

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 点击扭蛋机或底部按钮触发动画
3. 动画播放完成后自动恢复静态显示

## 浏览器支持

- Chrome/Edge (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- 移动端浏览器

## 自定义说明

### 修改活动日期
编辑 `index.html` 中的日期范围：
```html
<div class="date-range">12/19 – 12/25</div>
```

### 修改活动标题
编辑 `index.html` 中的标题：
```html
<span class="title-text">喜乐贩卖机</span>
```

### 更换动画视频
替换 `niudanji.mp4` 文件，保持文件名不变即可。

### 修改颜色主题
编辑 `styles.css` 中的颜色变量：
- 主色调：`#ff6b6b`, `#ff4757`
- 背景色：`#fafafa`, `#ffffff`

## 技术栈

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript

## 注意事项

- 确保 `niudanji.mp4` 文件存在且可访问
- 视频格式建议使用 MP4 (H.264 编码)
- 建议视频时长控制在 3-5 秒以内
